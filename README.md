# NikiBhavi — Malaysia Life Guide for Indians

🇲🇾 A comprehensive web guide for Indians living in or moving to Malaysia.

**Live:** https://nikibhavi.msdevbuild.com

## Features
- 17 guide pages covering visa, housing, banking, transport, food & more
- 14 financial & life tools (Tax calc, EPF, Remittance, Salary comparison, etc.)
- Live MYR/INR exchange rates

## Stack
React 18 + Vite 5 + CSS Modules

## Dev
```bash
npm install
npm run dev
```

## Deploy
Automatically deploys to GitHub Pages on push to `main` via GitHub Actions.

Manual deploy:
```bash
npm run deploy
```
